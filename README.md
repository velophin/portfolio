# 포트폴리오 웹사이트

개인 이력서 및 포트폴리오를 보여주는 반응형 웹사이트

## 🚀 주요 기능

- 반응형 디자인 (Desktop, Tablet, Mobile)
- 한국어 기반 UI/UX
- 부드러운 스크롤 애니메이션
- 모바일 햄버거 메뉴

## 🛠️ 기술 스택

- **Frontend**: HTML5, CSS3, JavaScript
- **디자인**: CSS Grid, Flexbox
- **아이콘**: Font Awesome
- **폰트**: Noto Sans KR

## 📁 프로젝트 구조

```
portfolio-website/
├── index.html          # 메인 HTML 파일
├── css/
│   └── style.css       # 스타일시트
├── js/
│   └── script.js       # JavaScript 기능
├── images/             # 이미지 파일들
└── README.md
```
